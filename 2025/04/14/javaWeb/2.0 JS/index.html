一、JavaScript 简介

JavaScript 是网页的编程语言，主要用于：

  • 控制网页中的 HTML 元素
  • 响应用户操作（按钮、键盘、鼠标等）
  • 动态加载数据（AJAX、Fetch）
  • 控制多媒体、动画、游戏等

运行环境：

  • 浏览器（如 Chrome、Firefox、Edge）
  • Node.js（运行在服务器端）
